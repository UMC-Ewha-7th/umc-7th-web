inline inline inline inline: width, height 속성 지정

inline 요소는 본인 크기 만큼만 차지하며, width, height 속성을 지정할 수 없다.

3. inline-block 요소 사용

inline-block 요소는 본인 크기 만큼만 차지하며, width, height 속성을 지정할 수 있다.
